# shell
## introduction
This shell is a command-line interpreter developed in C specifically for the Unix environment


## compile 
> make

## funtinons
* built-in command
    * echo
    * export
    * pwd
    * cd
    * history
    * jobs
    * kill
* signal handle
    * SIGINT
    * SIGTSTP
    * SIGCHLD
* execute program
